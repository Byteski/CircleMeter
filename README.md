# CircleMeter
A simple animated hardware meter on tray!

## Note
- This was made by using WinForms.
- .Net 8.0 is required, but it's included for now.

## Features
- Select between CPU/GPU/Memory and monitor them.
- Values can be changed between Usage & Temperature.
- The fast the color spins, lighter your CPU/GPU/Memory is.

## Known Issues (Current)
- CPU Temperature can't be measured.
- Memory doesn't support Temperature.

## Upcoming Features (Note)
- Add a home menu and separate menus as submenu.
- Add additonal feature where you can invert circle's speed
(It means the slower the color spins, lighter your CPU/GPU/Memory is)
